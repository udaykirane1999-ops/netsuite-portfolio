# UAT, Deployment & Release Governance

## Business Context
Frequent system enhancements and integrations required controlled testing and deployment.

## My Role
I coordinated UAT, defect tracking, and production deployments.

## Key Responsibilities
- Created UAT test scripts and managed test cycles
- Tracked defects and enhancements using Jira
- Coordinated sandbox to production deployments
- Prepared release notes and change logs
- Supported business sign-off before go-live

## Tools & Features
Jira, NetSuite Sandbox, SuiteBundler, UAT Scripts

## Outcome
Deployments were predictable and low-risk, ensuring system stability and user confidence.
