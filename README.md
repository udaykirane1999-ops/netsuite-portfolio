# NetSuite Portfolio — Uday Kiran Elati

NetSuite Functional Consultant / Administrator with 5+ years of experience supporting implementations, enhancements, integrations, and production support. - Data migration (CSV import planning, validation, rollback)
- UAT and release governance (test plans, sign-off, deployment checklists)
